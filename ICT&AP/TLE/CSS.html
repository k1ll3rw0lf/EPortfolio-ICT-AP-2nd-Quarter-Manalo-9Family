INTRO TO CSS

What I Learned

CSS is a styling language used to control the design and layout of web pages.
It works alongside HTML, while HTML structures the content, CSS makes it visually appealing and organized.
With CSS, web designers can change colors, fonts, spacing, borders, and even create animations and responsive layouts that adapt to different screen sizes.

Importance of CSS

CSS is one of the three core technologies of the web, along with HTML and JavaScript. It plays a vital role in:

Design and Layout
CSS allows you to create attractive web pages with colors, backgrounds, and different text styles.

Consistency
A single CSS file can control the design of multiple pages, ensuring a uniform look across a website.

Accessibility
CSS helps make content easier to read and navigate, improving the experience for users with disabilities.

Maintenance
Instead of editing every HTML file, you can change one CSS file to update your entire siteâ€™s appearance.

Performance
Well-written CSS can make websites load faster by reducing unnecessary styling and HTML code.

The Three Layers of a Webpage

Every webpage has three main layers that work together to form a complete web experience:

Structure Layer (HTML)
Defines the content and layout of a page.
It includes elements like headings, paragraphs, images, and links.

Presentation Layer (CSS)
Controls the design, colors, fonts, and positioning of HTML elements.
Makes the webpage more attractive and readable.

Behavior Layer (JavaScript)
Adds interactivity and dynamic features to a webpage.
Used for animations, form validation, and responding to user actions.

Together, these layers ensure a website is functional, attractive, and interactive
