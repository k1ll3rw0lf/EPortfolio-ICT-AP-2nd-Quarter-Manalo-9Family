CLASSES

What I Learned

Classes are a very important and effective way of targeting a specific group of elements in your webpage. Classes are user-defined selectors used to style any element. You can name classes anything, so feel free to choose any name that you can remember. Classes start with a period (.) and then the class name. The class attribute is used inside the opening tag of an element to apply the class modifications to that element.

Classes are very helpful in making web pages and especially in modifying different elements at the same time.
