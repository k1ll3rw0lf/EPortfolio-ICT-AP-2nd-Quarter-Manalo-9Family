STYLESHEETS

What I Learned

CSS has many methods of applying styles to your webpage. These methods are called the stylesheets. There are 3 stylesheets that CSS has to offer
  1. Inline CSS
  2. Internal/Embedded CSS
  3. External/Linked CSS

Inline CSS

Inline CSS is a stylesheet that involves the style attribute.
It allows you to apply the styles in the html tags themselves.
It has the highest priority in terms of application. This means that if you use Internal CSS, External CSS, and Inline CSS on the same element, Inline CSS will take over.

Internal CSS

Internal CSS is a stylesheet that involves the style tag.
This stylesheet allows you to apply styles to the webpage itself. In one way or another, you can apply the style to anything including the body tag or even a specific paragraph
It has the second highest priority so it overrides external CSS changes but it does not override inline CSS changes.
It is also placed inside the head tag.

External CSS

External CSS is a stylesheet that involves using an external stylesheet and a link tag to apply CSS to multiple webpages.
It is very effective as you only have to write the code once and you can just link the CSS file using the link tag.
