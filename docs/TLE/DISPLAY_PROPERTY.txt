<h1>DISPLAY PROPERTY</h1>
<h2>What I Learned</h2>
There are 2 types of elements, block level elements and inline level elements.<br>
Block level elements are elements that stack vertically while inline level elements stack horizontally.<br>
Some notable examples for each are: The list elements, paragraph element and heading elements for block level and anchor element, span element, and strong element for inline level.<br><br>
How an element behaves when styled with CSS will change based on the display mode of an element. This is why we have to be careful with how we make our codes. <br><br>One key thing to remember is that <strong>Inline level elements cannot contain block level elements.</strong><br><br>
Different values for the display property affect how elements appear. The values you can use are inline, block, list-item, none, and inherit.
